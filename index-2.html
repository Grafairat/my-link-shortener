⚙ Настройка базы данных

Нужно сделать один раз. Займёт ~3 минуты.
1

Зарегистрируйся на JSONBin.io

Зайди на jsonbin.io → нажми Sign Up → создай аккаунт (бесплатно).

2

Получи API Key

После входа нажми на иконку профиля вверху справа → API Keys → скопируй Master Key (начинается с $2b$...).

3

Создай Bin (хранилище)

На главной странице JSONBin нажми + New Bin → в поле введи {"links":[]} → нажми Create Bin. Скопируй Bin ID из адресной строки или из заголовка (длинный код после последнего /).

4

Вставь ключи сюда
